# 📖 Описание

**x0conf_dork** — это консольное приложение на Python, которое помогает быстро создавать точные поисковые запросы (Google Dorks) для поиска информации о человеке, организации или домене.

Инструмент поддерживает **21 категорию** поисковых запросов и позволяет автоматически открывать готовые Google-запросы в браузере.

Проект предназначен для образовательных целей, изучения поисковых операторов и проведения исследований исключительно в рамках законодательства.

# 📦 Установка

## Клонирование


# 🧭 Использование

После запуска появится главное меню.

```text
Выберите номер категории (0 для выхода): 1
```

Введите необходимые параметры.

Например:

```text
Имя и фамилия:
Иван Петров
```

После этого программа сгенерирует список Google-запросов.

Например:

```text
1. "Иван Петров" site:vk.com

2. "Иван Петров" site:ok.ru

3. "Иван Петров" site:instagram.com
```

Далее можно:

| Команда | Действие |
|----------|----------|
|n|Открыть выбранный запрос|
|a|Открыть все запросы|
|n|Вернуться в меню|
|q|Выход|

# 💡 Советы

- Используйте кавычки (`" "`) для точного поиска.
- Комбинируйте разные категории.
- Используйте исключения (`-site:` и `-filetype:`) для уменьшения количества мусора.
- Категория поиска по домену автоматически генерирует возможные корпоративные email.
- Поиск документов (`filetype:`) помогает находить открытые PDF, DOC и XLS.
